# osTicket Help Desk Ticketing System

## Project Overview

This project demonstrates the deployment and configuration of an osTicket Help Desk Ticketing System using XAMPP, Apache, PHP, and MySQL.

## Technologies Used

- Windows 11
- XAMPP
- Apache
- PHP 8.2
- MySQL
- phpMyAdmin
- osTicket 1.18.4

## Skills Demonstrated

- Help Desk Administration
- Database Management
- Troubleshooting
- Web Server Configuration
- System Administration
